## 🚀 Featured Projects

### 📉 Customer Churn Prediction System
> End-to-end ML pipeline for telecom customer churn prediction — deployed live on Streamlit Cloud.

| Detail | Value |
|--------|-------|
| 📁 Dataset | IBM Telco Customer Churn — 7,043 records, 20 features → 29 after encoding |
| 🤖 Models Evaluated | 8 (LR, Random Forest, Decision Tree, KNN, AdaBoost, Gradient Boosting, SVM, XGBoost) |
| 🏆 Final Model | XGBoost — Accuracy: **78.6%** \| Precision: **63.9%** \| ROC-AUC: **0.84** |
| ⚙️ Tuning | GridSearchCV with 5-fold CV — 720 fits |
| 🌐 Deployment | Streamlit Cloud — real-time predictions + business retention recommendations |

**Key churn drivers identified:** High monthly charges • Short tenure • Month-to-month contracts • Fiber optic subscriptions

**Tech Stack:** Python • Pandas • NumPy • Scikit-learn • XGBoost • Streamlit • Matplotlib • Seaborn
